## 一、个人信息资产清单
| 数据项 | 所在系统 | 存储形态 | 保护措施 | 保留策略 |
| ---- | ---- | ---- | ---- | ---- |
| 账号/显示名 | IdP | 明文列 | 访问控制+审计 | 账号生命周期 |
| 手机号 | IdP | 信封密文+HMAC 盲索引 | 密文自描述,索引不可逆 | 同上 |
| 口令 | 各系统 | 单向哈希(argon2id/PBKDF2-SM3) | 透明重哈希升级 | 不可导出 |
| TOTP 秘密 | IdP/certvault | 信封密文 | 同信封策略 | 解绑即删 |
| 证件影像 | certvault | 文件信封密文 | AAD 绑定属主 | 删除连带销毁 |
| 溯源笔记/定位 | certvault | 信封密文 | 属主/管理员可见 | 随记录 |
| 设备口令 | nvr | 信封密文 | 同信封策略 | 设备退库即删 |
| 学习记录 | quiz | 账号/游客 ID 关联 | 迁移码零个人信息 | 用户可清空 |

## 二、处理活动登记(模板)
- 处理目的:____ 法律依据:____(履行合同/法定义务/同意)
- 数据最小化说明:仅采集上表数据项,无画像/无对外提供。
- 委托处理:无(全部内网自处理);跨境:无。

## 三、影响评估要点(PIA 模板)
1. 风险识别:泄露/篡改/滥用场景与影响等级。
2. 已有措施映射:信封加密/审计链/最小授权/备份加密保管。
3. 剩余风险与处置:____(负责人/期限)。
4. 结论与签署:____
## 四、个人权利响应路径
- 查询/更正:管理台按账号检索;删除:账号注销联动密文销毁;
  响应时限与登记表:由运营方按制度执行并留档。
