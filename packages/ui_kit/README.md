# ui-kit(前端共享资产内部包 · 接口约定,H11 §一)

> 与后端共享库 sso_client/policy/crypto 对等的前端内部包(H01 §三)。
> 本里程碑仅落接口约定;实现随里程碑 9(前端三形态)交付。

## 必须提供的共享资产(H11 §五)
- **统一 fetch 封装**:自动携带 CSRF 令牌;生成/透传 X-Request-Id;
  统一处理 401(跳登录保留站内 next)/403/423(锁定,显示等待时长)/429(限速,
  显示等待时长);解析 AdapterResult 信封与字段级校验错误;
- **DEMO 横幅组件**:全站红色"演示模式,仅限测试" / 生产模式指示(05-D9);
- **手机号打码组件**(H04 §七);
- **登录态与错误处理**、基础组件、CSS 设计变量(F1 零 JS 页仅复用设计变量)。

## 红线(H11 §四,逐条 CI/E2E 固化)
- 禁止令牌入 localStorage/sessionStorage(certvault JWT 例外:仅存内存);
- 全部静态资源本地化同源,禁运行时 CDN/外网字体/统计脚本;
- E3 双条款:脚本引用元素先于 script 存在;[hidden]{display:none} 兜底。
